Bellabeat Case Study - Proyecto Completo de Portfolio Data Analyst

**Objetivo: **Desarrollar un proyecto profesional de análisis de datos utilizando el dataset Fitbit para generar recomendaciones estratégicas para Bellabeat.

# Resumen Ejecutivo

Problema de negocio.

Stakeholders.

Objetivos SMART.

Preguntas clave de análisis.

Hipótesis iniciales.

# Fase 1 - Pregunta (Ask)

Definir el problema de negocio.

Identificar stakeholders.

Definir KPIs: pasos, calorías, horas de sueño, usuarios activos.

Documentar hipótesis de negocio.

Definir criterios de éxito.

# Fase 2 - Prepara (Prepare)

Descargar y catalogar todos los CSV.

Crear diccionario de datos.

Evaluar calidad mediante ROCCC.

Identificar relaciones entre tablas.

Diseñar estructura de carpetas del proyecto.

# Fase 3 - Procesa (Process)

Limpieza de duplicados.

Tratamiento de nulos.

Conversión de tipos de datos.

Creación de variables derivadas.

Integración de tablas.

Generación de dataset analítico.

# Fase 4 - Analiza (Analyze)

Análisis descriptivo.

Análisis de actividad física.

Análisis de sueño.

Correlaciones.

Segmentación de usuarios.

Clustering con K-Means.

Insights de negocio.

# Fase 5 - Comparte (Share)

Dashboard Power BI.

Storytelling ejecutivo.

Presentación para stakeholders.

Publicación en GitHub.

Publicación en LinkedIn.

# KPIs Recomendados

Pasos medios diarios.

Calorías medias.

Horas medias de sueño.

Usuarios activos por segmento.

Tasa de adopción del dispositivo.

Tiempo sedentario promedio.

# Visualizaciones Recomendadas

Histograma de pasos.

Boxplot de calorías.

Heatmap de sueño.

Scatter pasos vs calorías.

Scatter actividad vs sueño.

Dashboard ejecutivo.

# Estructura GitHub

README profesional.

Notebook documentado.

Dataset limpio.

Scripts SQL.

Capturas del dashboard.

Presentación final.

# Cronograma de 14 días

Día 1-2: Pregunta.

Día 3-4: Preparación.

Día 5-7: Procesamiento.

Día 8-11: Análisis.

Día 12-14: Dashboard y presentación.

# Checklist Final de Portfolio

Repositorio GitHub completo.

Dashboard interactivo.

Presentación ejecutiva.

Publicación LinkedIn.

Conclusiones y recomendaciones.

# Recomendaciones de Negocio Esperadas

Gamificación para aumentar actividad.

Segmentación de campañas por nivel de actividad.

Notificaciones personalizadas.

Promoción de funcionalidades relacionadas con el sueño.

Programas de fidelización para usuarias activas.
